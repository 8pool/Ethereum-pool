How to connect

Stratum mode using Genoils Miner
Download, configure and run Genoils Miner
  • Windows: Windows Builds
  • Source / Linux: Build instructions
Then start your miners by putting the following commands into a bat file
setx GPU_FORCE_64BIT_PTR 0
setx GPU_MAX_HEAP_SIZE 100
setx GPU_USE_SYNC_OBJECTS 1
setx GPU_MAX_ALLOC_PERCENT 100
setx GPU_SINGLE_ALLOC_PERCENT 100
ethminer.exe --farm-recheck 200 -G -S eu1.ethpool.org:3333 -FS us1.ethpool.org:3333 -O <Your_Ethereum_Address>.<RigName>
This will use the eu1 server as main server and the us1 server as backup server
Choose from the following stratum servers:
North America (East): us1.ethpool.org:3333
North America (West): us2.ethpool.org:3333
Europe: eu1.ethpool.org:3333
Asia: asia1.ethpool.org:3333

Installation guide
Stratum mode using QtMiner
Windows
  • Download QtMiner: qtminer.zip
  • Extract the zip archive
  • Start QtMiner using the following command: qtminer.exe -u <Your_Ethereum_Address>.<RigName> -G
  • Full example:
    North America (East): qtminer.exe -s us1.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G
North America (West): qtminer.exe -s us2.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G
Europe: qtminer.exe -s eu1.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G
Asia: qtminer.exe -s asia1.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G
Ubuntu
  • Download QtMiner:
    • Ubuntu 14.04: qtminer.tgz
    • Ubuntu 15.04: qtminer_1504.tgz
  • Extract the archive: tar zxvf qtminer.tgz
  • Make the qtminer.sh script executable: chmod +x qtminer.sh
  • Start QtMiner using the following command: ./qtminer.sh -u <Your_Ethereum_Address>.<RigName> -G
  • Full example: 
    North America (East): ./qtminer.sh -s us1.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G
North America (West): ./qtminer.sh -s us2.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G
Europe: ./qtminer.sh -s eu1.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G
Asia: ./qtminer.sh -s asia1.ethpool.org:3333 -u 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1 -G

Source: http://github.com/etherchain-org/qtminer

Getwork mode using ethminer

Mining using the standard ethminer is no longer recommended. Please use one of the available stratum enabled miners!

You need to have a working ethminer installation which comes with the c++ eth version of Ethereum. For installation instructions see ethereum.org/cli.

To join the pool start up ethminer using the -F flag and point it to the following address

http://us1.ethpool.org/miner/<account>.<worker>/<hashrate>

<account> is your ethereum account address where the pool payments will be send to.

<hashrate> is the difficulty of the work packages you will receive from the pool. Set it by providing the hashrate of your ethminer instance in MH/s.

Full example to mine with a 10MH/s ethminer instance to address 0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01:

ethminer -F http://us1.ethpool.org/miner/0x4bb96091ee9d802ed039c4d1a5f6216f90f81b01.rig1/10 -G (+ your usual options)

If you receive frequent "JSON-RPC Problem" errors try to increase your --farm-recheck value.

You should find a valid share every 1-2 minutes. If it constantly takes you longer to find a share try to lower your hashrate values (minimum is 0.1).
